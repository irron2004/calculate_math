## 1) 핵심 의사결정 미완료(ALL/ANY, 노드 노출, 보스 기준)
- 입장: 부분 수용.
  - Unlock 규칙: Phase 1은 ALL 고정, 제한적 ANY는 교사 모드 + Phase 2.
  - 보스전: 합격선 80/100, 쿨다운 없음(Phase 1). 실패 시 리미디얼 추천 필수.
  - 학년 범위 밖 노드: 기본 흐림 처리 + 보호자/교사용 스위치로 진하게 보기.
- 추가 조사: NONE(내부 의사결정 완료).
- 수정사항: PRD v1.0.1에 반영, `docs/pm/DECISIONS.md` 기록.

## 2) 타겟 사용자 & JTBD 범위
- 입장: 수용(Phase 1 축소). 반박: 중등 노출의 ‘동기부여’ 가치는 인정하되 기본은 흐림 처리.
- 추가 조사: 사용자 반응 정성 조사로 ‘흐림 노출’의 적절성 검증.
- 수정사항: Phase 1 타겟을 초2~초4로 축소, 중등 콘텐츠는 Phase 2로 이관. PRD 반영.

## 3) UX 플로우 보완(세션 종료/보스 실패)
- 입장: 수용.
- 추가 조사: 애니메이션 밀도/접근성 영향에 대한 간단한 유저 테스트.
- 수정사항:
  - 세션 종료 → “새로 해금된 스킬” 애니메이션 + “다음 추천” 자동 포커스 플로우 추가.
  - 보스 실패 → 부족 AS 강조 + “이 스킬부터 연습” CTA 명시. PRD/Spec 반영.

- 수정사항(Phase 1 최소 기준):
  - 세션 내 중복 방지(seed 블랙리스트/최근 히스토리 체크).

  - IRT는 Phase 2로, 파일럿 수집 계획 PRD에 명시.
