# Product Decision Log

본 문서는 제품 핵심 의사결정을 기록하는 단일 로그입니다. 각 항목은 ID, 상태, 근거, 반영 버전/일자를 포함합니다.

## D-0001 Unlock 규칙 (ALL vs ANY)
- 상태: 확정(Phase 1 = ALL 고정, 제한적 ANY는 교사 모드 + Phase 2)
- 근거: 구현 단순성, 오판 리스크 제거, 교사 주도 시나리오 보존
- 반영: PRD v1.0.1, 2025-11-10

## D-0002 보스전 합격선/쿨다운
- 상태: 확정(80/100, 쿨다운 없음)
- 근거: 회귀 연습-재도전 루프 장려, 실패 시 리미디얼 추천 제공
- 반영: PRD v1.0.1, 2025-11-10

## D-0003 학년 범위 밖 노드 노출
- 상태: 확정(기본 흐림, 보호자/교사 스위치로 진하게 보기)
- 근거: 동기부여 효과와 부담감의 균형
- 반영: PRD v1.0.1, 2025-11-10

## D-0004 Phase 1 타겟/범위/일정
- 상태: 확정(초2~초4, 코스 3개×S1/S2, 템플릿 30개, 10–12주)
- 근거: 리소스/리스크 기반 현실화
- 반영: PRD v1.0.1, 2025-11-10

## D-0005 설명 채점 범위
- 상태: 확정(Phase 1 선택, S3에서만 필수; 키워드 매칭)
- 근거: 구현 난이도·효용 균형, 러닝 부하 최소화
- 반영: PRD v1.0.1, 2025-11-10

## D-0006 접근성 대체 UI
- 상태: 확정(리스트 뷰 제공, 키보드/리더 최적화)
- 근거: WCAG AA 현실성 확보
- 반영: PRD v1.0.1, 2025-11-10

## D-0007 수익화 (무료/유료 경계)
- 상태: 확정(무료=S1 무제한 + S2 일부, 유료=전체+리포트+교사 콘솔)
- 근거: 핵심 가치(전이) 체험 유지 + 유료 전환 계기 제공
- 반영: PRD v1.0.1, 2025-11-10
