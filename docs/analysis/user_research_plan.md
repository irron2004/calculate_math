# 사용자 리서치 플랜(초등 저학년/보호자)

목표: 디아블로식 스킬 트리/리스트 뷰/흐림 노출의 수용성과 전환 모달 반응 검증.

## 대상/샘플
- 보호자 5–10명, 초2~초4 학생 5–10명(보호자 동반)

## 과업
- 트리 vs 리스트 초기 탐색(과업 성공/탐색 경로/발화)
- 흐림 노드 노출 반응(부담/동기부여)
- S2 2번째 완료 시 전환 모달 반응
- 세션 종료 → “새로 해금/다음 추천 자동 포커스” 플로우 사용성
- 보스전 실패 화면에서 리미디얼 제안 클릭률/재도전 의사 확인

## 측정
- 과업 완료율/시간/오류, SUS 10문항, 간단 인터뷰(5문)
- 자동 포커스 인지 여부, 리미디얼 CTA 클릭률/재도전율

## 산출물
- 인사이트 요약, 개선안, PRD 반영 포인트 목록
