# 경쟁 분석 플랜

목표: 주요 경쟁 제품의 스킬트리/진행도/과금/접근성/콘텐츠 운영을 벤치마크하여 차별화 포인트 검증.

## 후보
- Khan Academy Kids / Khan Academy
- Prodigy Math
- Photomath
- IXL

## 분석 축
- 스킬 표현(그래프/리스트), 전이 개념 유무
- 문제 생성/변형/난이도 보정(IRT)
- 수익화 모델과 전환 유도 타이밍
- 접근성/저사양 UX(키보드/리더/저대역폭)

## 산출물
- 기능 비교 매트릭스, 차별화 요약 1p, 리스크/기회 항목
- 가격/전환 트리거 스냅샷(무료 한계 지점·업셀 문구) 캡처
- 접근성 모드(키보드/리더) 지원 유무 체크리스트
