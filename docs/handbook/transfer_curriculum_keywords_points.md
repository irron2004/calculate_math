본 문서는 이전 메시지에서 정리한 12개 전이(Transfer) 커리큘럼에 대해, 각 항목별로 S1(초2 표현)·S2(브리지)·S3(최상위) 단계에서 가르칠 키워드와 교수 포인트를 정리합니다. 문서 끝에는 완주 시 익히는 개념(트리 노드 기준 + 일반적 학년 대역)을 한눈에 보는 커버리지 요약 표를 포함합니다.

참고: 학년 매핑은 한국 학교 교육과정의 전형적 흐름을 바탕으로 한 권장 대역이며, 지역·개정에 따라 다를 수 있습니다. 우리는 트리 노드(예: ALG4 일차함수)를 기준으로 정확하게 레퍼런스합니다.

---

## 1) 비율·비례 전이 (곱셈/나눗셈 → 비율·비례 → 반비례·유리함수) 〔렌즈: ➗Ratio〕

핵심 한 줄: “두 양의 비율이 일정하면 한쪽이 k배일 때 다른 쪽도 k배다.”

S1(초2 표현) – 키워드
- 배합, 묶음, 1대2, 단위량(한 컵 기준), 같은 맛/같은 색
- 표로 비율 채우기, 그림 아이콘(컵/블록)
  교수 포인트
- “같은 만큼 더하기”가 아니라 “같은 배율”을 말로 구분
- 단위율(1의 값)을 말로 설명: “컵 1개당 물은 몇 컵?”
- 같은 비율/다른 차이를 대비 제시

S2(브리지) – 키워드
- 비례표, 단위율, (y=kx), 원점을 지나는 직선
  교수 포인트
- 비례식에서 k(비례상수) 찾기(표→식→그래프 전환)
- 단위율을 기울기의 뜻으로 연결

S3(최상위) – 키워드
- 반비례 (xy=c), 유리함수 스케치, 속력×시간 일정
  교수 포인트
- “곱이 일정”이라는 문장 → 표 완성 → 간단 그래프 형태 인식
- 비례 vs 반비례 상(相) 대비(직선/하이퍼볼릭)

오개념 포인트: 비율과 차이 혼동 / 단위율 미해석
전이 맥락: 레시피·페인트 배합 / 운임표 / 넓이 일정 직사각형

---

## 2) 지수 성장 전이 (곱셈 → 등비수열 → 지수·로그) 〔렌즈: 📈Scale〕

핵심 한 줄: “매 단계 배수로 늘어난다 → (a·r^{n}).”

S1 – 키워드
- 두 배/세 배, 하루마다 2배, 표 채우기
  교수 포인트
- 등차 vs 등비 표/그래프를 나란히 비교(“+k” vs “×k”)

S2 – 키워드
- 등비수열 일반항, 베이스 r, 로그 눈금(간격은 같아도 값은 배수)
  교수 포인트
- “몇 단계 뒤 몇 배?”를 말로 쓰고 식으로 옮기기

S3 – 키워드
- 지수함수/로그함수 상호변환, (n=log_r(A/A_0)) 구조
  교수 포인트
- 성장/감쇠 시나리오에서 시간-배수-지수식 연결

오개념: +k/×k 혼동 / 로그를 지수 반대로만 기계 암기
전이: 이자/세균/바둑돌 더블링/데시벨

---

## 3) 닮음·삼각비 전이 (측정 → 닮음 → 삼각비/삼각함수) 〔렌즈: 📏Scale〕

핵심 한 줄: “각이 같으면 길이 비가 일정하다.”

S1 – 키워드
- 확대/축소, 축척, 비례 자르기
  교수 포인트
- 길이비와 넓이비 구분(길이 2배 → 넓이 4배 시각화)

S2 – 키워드
- 직각삼각형, (sin, cos, tan) = 변의 비
  교수 포인트
- 기울기를 (tan)의 생활어로 연결(경사/사다리)

S3 – 키워드
- 그림자 측정, 역삼각비 상황, 간단 응용
  교수 포인트
- 실거리 추정에서 오차 원인(그림자, 지면 경사) 토론

오개념: 길이비↔넓이비 혼동 / 각 유지 조건 무시
전이: 지도축척/그림자/경사로

---

## 4) 적분 직관 전이 (막대 합 → 면적 누적 → 정적분 의미) 〔렌즈: ⬛Accumulation〕

핵심 한 줄: “부분들의 합이 전체(면적·양)를 이룬다.”

S1 – 키워드
- 막대높이 합, 폭=1, 직사각형 쪼개기
  교수 포인트
- “높이 합”이 아니라 높이×폭의 합임을 말로 강조

S2 – 키워드
- 사다리꼴 근사, (∑) 기호, 연속 곡선 아래 넓이
  교수 포인트
- 계단→사다리꼴로 근사 정교화

S3 – 키워드
- 정적분 해석(기호보단 의미 중심), 두 함수 사이 면적 비교
  교수 포인트
- 동일 데이터로 합과 면적 두 표현 왕복

오개념: 폭 개념 누락 / 축 단위 무시
전이: 수도 사용량/걸음 합/계단 면적

---

## 5) 변화율 전이 (거리–시간 → 평균변화율 → 등가속도/미분 직관) 〔렌즈: 🔺Difference〕

핵심 한 줄: “(Δy/Δx) = 기울기 = 평균변화율.”

S1 – 키워드
- 10분 간격 이동거리, 기울기 말로 설명
  교수 포인트
- 같은 y값 vs 같은 기울기 대비 사례

S2 – 키워드
- 속도–시간 그래프, 면적=거리(누적)
  교수 포인트
- 차분(기울기) ↔ 누적(면적) 렌즈 교차

S3 – 키워드
- 등가속도, 순간변화율 직관(짧은 구간 한계)
  교수 포인트
- 평균→순간으로 창 좁히기(구간 줄이기) 활동

오개념: 기울기와 y값 혼동 / 축 단위 미고려
전이: 자전거 주행/온도 변화/저수지 수위

---

## 6) 조합·확률 전이 (경우의 수 → 확률 → 이항분포/정규근사) 〔렌즈: 🎲Randomness〕

핵심 한 줄: “전체 경우 중 성공 경우의 비율이 확률.”

S1 – 키워드
- 트리 다이어그램, 배열 놀이
  교수 포인트
- 배반/포함 상황을 그림으로 구분

S2 – 키워드
- (C(n,k)) 표, (p^k(1-p)^{n-k})
  교수 포인트
- “몇 번 중 몇 번 성공”을 말로→식으로

S3 – 키워드
- 대수의 법칙 직관, 종 모양(정규근사)
  교수 포인트
- 큰 n에서 대칭/봉우리의 이유 설명

오개념: 중복계수 / 독립성 혼동
전이: 주사위 합/시험 OX/랜덤 상자

---

## 7) 변동성·정규 전이 (평균 → 편차 → 분산·표준편차 → 정규분포) 〔렌즈: ⬛Accumulation + 🔺Difference〕

핵심 한 줄: “중심에서 얼마나 퍼졌는가가 표준편차.”

S1 – 키워드
- 평균선, 편차(위·아래), 스티커 그래프
  교수 포인트
- 절댓값 편차로 직관 먼저

S2 – 키워드
- 분산/표준편차, 상자그림
  교수 포인트
- 제곱 단위→루트로 단위 복귀의 의미

S3 – 키워드
- 68–95–99.7 규칙, 퍼짐 비교
  교수 포인트
- 같은 평균 다른 표준편차 대비

오개념: 단위 혼란 / 이상치에 민감도 오해
전이: 반 모둠 키/기록/센서 노이즈

---

## 8) 해석기하 전이 (좌표놀이 → 직선/원 방정식 → 포물선) 〔렌즈: 🔄Transform〕

핵심 한 줄: “도형의 성질을 식으로 바꾼다.”

S1 – 키워드
- 격자 좌표, 두 점 거리, 중점
  교수 포인트
- 좌표 간 차이로 거리 직관

S2 – 키워드
- 직선 (y=mx+b), 원 ((x-a)^2+(y-b)^2=r^2)
  교수 포인트
- 기울기=변화량, 절편=시작값 언어화

S3 – 키워드
- 포물선: 초점-준선 정의, 기본식
  교수 포인트
- 정의→그림→식 유도 흐름

오개념: 기울기·절편 의미 혼동 / 거리공식 단순 암기
전이: 길찾기/반경/반사

---

## 9) 수론 전이 (약수·배수 → GCD → 유클리드 → 모듈러) 〔렌즈: 🔺Difference + 🔄Transform〕

핵심 한 줄: “나눗셈의 공통 구조로 최대공약수를 찾는다.”

S1 – 키워드
- 묶음 나누기, 나머지, 배수/약수
  교수 포인트
- 동시 배분과 남는 것(나머지) 시각화

S2 – 키워드
- 유클리드 알고리즘, 나머지 연산
  교수 포인트
- 큰 수→작은 수로 반복하여 줄이는 아이디어

S3 – 키워드
- 시계산(모듈러), 간단 암호 비유
  교수 포인트
- 합동 개념을 패턴 주기로 설명

오개념: 소수=작은 수 / 최대공약수·최소공배수 혼동
전이: 달력 주기/자리수 규칙/암호 퍼즐

---

## 10) 재귀 전이 (패턴 → 피보나치 → 점화식 → (선택) 행렬) 〔렌즈: 🔺Difference〕

핵심 한 줄: “앞의 값들로 다음 값을 만든다.”

S1 – 키워드
- 타일 규칙, 이전 것 합치기
  교수 포인트
- 초기값의 중요성 체험(바꾸면 전혀 달라짐)

S2 – 키워드
- (a_n=a_{n-1}+a_{n-2}), 폐쇄형 감각
  교수 포인트
- 재귀→패턴→성장 비율 언어화

S3 – 키워드
- (선택) 행렬 표현, 황금비 근사
  교수 포인트
- 재귀를 변환 규칙으로 보는 눈

오개념: 규칙 누락 / 초기값 무시
전이: 계단 오르기/토끼 번식/나선

---

## 11) 벡터·투영 전이 (화살표 합 → 내적 → 투영 → 최소제곱) 〔렌즈: 🔄Transform〕

핵심 한 줄: “방향으로의 그림자(투영) 크기를 내적이 준다.”

S1 – 키워드
- 화살표 합치기(머리-꼬리), 평행이동
  교수 포인트
- 크기 vs 방향을 말로 분리

S2 – 키워드
- 성분, 내적, 수선 거리
  교수 포인트
- 직선에 내린 그림자 길이 해석

S3 – 키워드
- 오차 최소, 직선에 데이터 투영
  교수 포인트
- “가장 가까운”을 투영으로 직관화(회귀의 씨앗)

오개념: 방향 무시 / 내적을 곱셈으로만 이해
전이: 바람 합력/경사로/간단 회귀

---

## 12) 도형 변환 전이 (평행이동·대칭 → 회전·합성 → 좌표변환) 〔렌즈: 🔄Transform〕

핵심 한 줄: “변환은 도형을 움직이는 규칙, 합성이 가능.”

S1 – 키워드
- 좌우/상하 대칭, 평행이동
  교수 포인트
- 축/중심을 항상 표시하고 말로 지시

S2 – 키워드
- 90°/180° 회전, 대칭·회전 합성
  교수 포인트
- 합성의 순서가 다르면 결과가 달라짐 체험

S3 – 키워드
- (입문) 좌표변환, 간단 행렬 이미지
  교수 포인트
- 변환을 규칙(함수)로 보는 관점 도입

오개념: 뒤집기·회전 혼동 / 합성 순서 무시
전이: 종이접기/로고/타일링

# 완주 시 익히는 개념 요약(트리 노드 + 일반적 학년 대역)

표의 “트리 노드”는 프로젝트 트리 ID/명칭을 따릅니다. “학년 대역”은 전형적 권장 범위입니다.

| 전이 스레드 | 주된 트리 노드(예) | 보조 트리 노드(예) | 일반적 학년 대역(권장) | 상위 교과 기여 |
| ------ | ----------------- | --------------- | --------------------- | --------- |
| 비율·비례 | ALG3 일차방정식/비례, ALG4 일차함수 | NUM2 분수/나눗셈, STA1 표/그래프 | 비례: 초6~중1 / 반비례: 중1 / 유리함수: 중3~고1 | 함수 개념 정착, 물리 비례 법칙 |
| 지수 성장 | ALG6 수열(등비), ALG 지수·로그함수 | CAL1 변화율 직관(성장 속도) | 등비: 고1 / 지수·로그: 고1~고2 | 지수 모델링, 로그 스케일 읽기 |
| 닮음·삼각비 | GEO3 닮음, GEO3/6 삼각비(기초) | ALG4 기울기=탄젠트 | 닮음: 초5~중1 / 삼각비: 중3~고1 | 측량, 물리 경사, 파형 해석 기초 |
| 적분 직관 | CAL4 적분법(의미), STA1 막대그래프 | CAL3 미분–적분 관계 직관 | 적분 의미: 고2 / 막대누적: 초3~중1 | 면적/누적량, 물리·통계 누적 |
