# 2022 개정 수학 교육과정 구조 분석 (조사 메모 정리)

이 문서는 **2022 개정 수학 교육과정**을 “트리/그래프 데이터”로 옮기기 위해, 구조(계층)와 코드(식별자) 패턴을 정리한 메모입니다.
- 목적: `학년(군) → 영역 → 단원 → 성취기준` 형태로 **노드/엣지 그래프 모델**을 만들기 위한 기준 정리
- 주의: 아래 내용은 조사 메모를 **정리/구조화**한 것으로, 최종 데이터는 **NCIC/교육부 고시 원문**으로 교차 확인 필요

---

## 1) 초등 수학(1~6학년) 큰 구조

### 1.1 학년군 체계
2022 개정 초등 수학은 학년을 3개 **학년군**으로 묶어서 내용을 제시합니다.
- 1~2학년군
- 3~4학년군
- 5~6학년군

### 1.2 영역(4개)
각 학년군은 수학 내용을 4개 영역으로 구분합니다.
- 수와 연산
- 변화와 관계
- 도형과 측정
- 자료와 가능성

### 1.3 계층(노드 구조) 제안
뷰어/검증기 데이터로 만들 때는 아래처럼 계층을 두면 구현이 단순해집니다.
- `학년군` → `영역` → `단원(교과서 주제)` → `성취기준`

각 노드는 최소한 아래 정보를 갖습니다.
- `id`: 노드 식별자(영역/단원/성취기준 코드 등)
- `title`: 노드 이름(예: “수와 연산”, “분수”, “성취기준 문장 요약”)
- `parent_id`: 상위 노드 id
- `grade_band`: 1-2 / 3-4 / 5-6
- (선택) `text`: 성취기준 원문 문장

---

## 2) 코드(식별자) 패턴 예시 (초등)

아래 예시는 조사 메모에서 사용된 패턴입니다.
- 영역(예): `2수01` (학년군+과목+영역)
- 단원(예): `2수01-A`
- 성취기준(예): `2수01-01`

영역 번호 `01~04`는 일반적으로 아래 4영역에 대응합니다.
- `01`: 수와 연산
- `02`: 변화와 관계
- `03`: 도형과 측정
- `04`: 자료와 가능성

---

## 3) 초등 1~2학년군 예시

### 3.1 수와 연산 (영역 ID: `2수01`)
- 영역: 수와 연산 (`2수01`)
  - 단원: 네 자리 이하의 수 (`2수01-A`)
    - 성취기준: `2수01-01` ~ `2수01-04`
  - 단원: 두 자리 수 범위의 덧셈과 뺄셈 (`2수01-B`)
    - 성취기준: `2수01-05` ~ `2수01-09`
  - 단원: 한 자리 수의 곱셈 (`2수01-C`)
    - 성취기준: `2수01-10` ~ `2수01-11`

### 3.2 변화와 관계 (영역 ID: `2수02`)
- 영역: 변화와 관계 (`2수02`)
  - 단원: 규칙 (`2수02-A`)
    - 성취기준 예: `2수02-01` (생활 주변 규칙 변화 표현 등)

### 3.3 도형과 측정 (영역 ID: `2수03`)
- 영역: 도형과 측정 (`2수03`)
  - 단원: 입체도형의 모양 (`2수03-A`) → `2수03-01`
  - 단원: 평면도형과 그 구성 요소 (`2수03-B`) → `2수03-02`
  - 단원: 양의 비교 (`2수03-C`) → `2수03-03`
  - 단원: 시각과 시간 (`2수03-D`) → `2수03-04`
  - 단원: 길이 측정 (`2수03-E`) → `2수03-05`

### 3.4 자료와 가능성 (영역 ID: `2수04`)
- 영역: 자료와 가능성 (`2수04`)
  - 단원: 자료의 분류 (`2수04-A`) → `2수04-01`
  - 단원: 표와 그림그래프 (`2수04-B`) → `2수04-02`, `2수04-03`

## 4) 초등 3~4학년군 예시

### 4.1 수와 연산 (영역 ID: `4수01`)
- 다섯 자리 이상의 수 (`4수01-A`) → `4수01-01`
- 분수 (`4수01-B`) → `4수01-02`
- 소수 (`4수01-C`) → `4수01-03`
- 세 자리 수의 덧셈과 뺄셈 (`4수01-D`) → `4수01-04`
- 자연수의 곱셈과 나눗셈 (`4수01-E`) → `4수01-05`, `4수01-06`
- 분모가 같은 분수의 덧셈과 뺄셈 (`4수01-F`) → `4수01-07`
- 소수의 덧셈과 뺄셈 (`4수01-G`) → `4수01-08`

### 4.2 변화와 관계 (영역 ID: `4수02`)
- 규칙 (`4수02-A`) → `4수02-01`
- 동치 관계 (`4수02-B`) → `4수02-02`
- 대응 관계 (`4수02-C`) → `4수02-03`

### 4.3 도형과 측정 (영역 ID: `4수03`)
- 도형의 기초 (`4수03-A`) → `4수03-01`
- 원의 구성 요소 (`4수03-B`) → `4수03-02`
- 여러 가지 삼각형 (`4수03-C`) → `4수03-03`
- 여러 가지 사각형 (`4수03-D`) → `4수03-04`
- 다각형 (`4수03-E`) → `4수03-05`
- 평면도형의 이동 (`4수03-F`) → `4수03-06`
- 시각과 시간(초) (`4수03-G`) → `4수03-07`
- 긴 길이/들이/무게/각도 측정 (`4수03-H`) → `4수03-08` ~ `4수03-10`

### 4.4 자료와 가능성 (영역 ID: `4수04`)
- 막대그래프 (`4수04-A`) → `4수04-01`
- 꺾은선그래프 (`4수04-B`) → `4수04-02`
- 평균 (`4수04-C`) → `4수04-03`

---

## 5) 초등 5~6학년군 예시

### 5.1 수와 연산 (영역 ID: `6수01`)
- 자연수의 혼합 계산 (`6수01-B`) → `6수01-01`
- 수의 범위와 어림 (`6수01-A`) → `6수01-02`, `6수01-03`
- 약수와 배수 (`6수01-C`) → `6수01-04`, `6수01-05`
- 분수의 덧셈과 뺄셈 (`6수01-D`) → `6수01-06` ~ `6수01-08`
- 나눗셈에서의 분수/소수 표현 (`6수01-H`) → `6수01-10`, `6수01-14`
- 분수의 곱셈과 나눗셈 (`6수01-E`) → `6수01-09`, `6수01-11`
- 분수와 소수의 관계 (`6수01-F`) → `6수01-12`
- 소수의 곱셈과 나눗셈 (`6수01-G`) → `6수01-13`, `6수01-15`

### 5.2 변화와 관계 (영역 ID: `6수02`)
- 비와 비율 (`6수02-A`) → `6수02-01`, `6수02-02`
- 비례식과 비례배분 (`6수02-B`) → `6수02-03`

### 5.3 도형과 측정 (영역 ID: `6수03`)
- 합동과 대칭 (`6수03-A`) → `6수03-01`, `6수03-02`
- 입체도형 (`6수03-B`) → `6수03-03` ~ `6수03-05`
- 도형의 측정 (`6수03-C`) → `6수03-06` ~ `6수03-08`

### 5.4 자료와 가능성 (영역 ID: `6수04`)
- 띠그래프와 원그래프 (`6수04-A`) → `6수04-01`
- 가능성 (`6수04-B`) → `6수04-02`

## 6) 중학교 수학(중1~중3) 큰 구조

### 6.1 영역(4개) 유지
중학교도 초등과 동일하게 4개 영역 체계를 유지합니다.
- 수와 연산 / 변화와 관계 / 도형과 측정 / 자료와 가능성

### 6.2 제시 방식(연계 과정)
조사 메모 기준으로, 중학교 수학은 “중1~중3”이 하나의 연계된 과정으로 제시되며(영역별로 성취기준 제시), 학교에서 학년별 배정을 하는 형태로 이해할 수 있습니다.

### 6.3 코드 패턴 예시
- 영역(예): `9수01`
- 성취기준(예): `9수01-01` ~

### 6.4 영역별 주요 내용(요약)
- `9수01` 수와 연산: 소인수분해/최대공약수·최소공배수, 정수·유리수 사칙연산, 순환소수, 제곱근·무리수·실수, 근호 포함 식 계산 등
- `9수02` 변화와 관계: 문자와 식, 일차방정식, 좌표/그래프, 정비례·반비례, 지수법칙 등
- `9수03` 도형과 측정: 기본 도형, 합동/닮음, 피타고라스 정리, 삼각비, 도형의 이동/대칭, 입체도형 등
- `9수04` 자료와 가능성: 대푯값(중앙값/최빈값 등), 자료 표현(상자그림/산점도 등), 경우의 수/간단 확률 등

## 7) 고등학교 수학 큰 구조(요약)

조사 메모 기준으로, 2022 개정 고등 수학은 고교학점제 흐름에 맞춰 공통과목과 선택과목으로 구성되며, 4영역 체계를 유지합니다.

### 7.1 공통과목(예)
- 공통수학 1: 다항식, 방정식과 부등식, 경우의 수, 행렬
- 공통수학 2: 도형의 방정식, 집합과 명제, 함수와 그래프

### 7.2 선택과목(예)
- 일반 선택: 대수, 미적분 I, 확률과 통계
- 진로 선택: 미적분 II, 기하, 실용 통계, 수학과 문화, 직무 수학 등

> 고등은 “과목 → 단원 → 성취기준” 구조가 강해져서, 데이터 모델에서도 `과목(course)` 노드를 명시적으로 두는 편이 자연스럽습니다.

## 8) 그래프 데이터로 옮길 때의 팁

### 8.1 권장 노드 타입(최소)
- `school_level`(초/중/고) *(선택)*
- `grade_band`(초등 1-2/3-4/5-6)
- `domain`(4영역)
- `unit`(단원)
- `standard`(성취기준)

### 8.2 필수 무결성 규칙(검증 포인트)
- ID 유일성
- parent 존재
- parent/children 양방향 일관성(선택적으로 children_ids를 들고 갈 경우)
- 루트에서 도달 가능한지(orphan)
- 사이클(cycle) 없음
